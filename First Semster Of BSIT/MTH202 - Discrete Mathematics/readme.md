# 📚 Course Overview

Discrete Mathematics focuses on mathematical structures that are fundamentally discrete rather than continuous. It includes logic, sets, relations, functions, combinatorics, graph theory, and probability.

This course builds the foundation for computer science, algorithms, and programming logic.
